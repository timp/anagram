To build the application and its documentation:

mvn clean install site

This creates the executable jar which can be executed as follows:

java -jar target/Anagrammer-1.0.jar

Or for more convenience on Unix:

chmod + x target/Anagrammer-1.0.jar
target/Anagrammer-1.0.jar

A bash script is supplied:

./build.sh
